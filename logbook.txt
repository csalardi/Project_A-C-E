[22.04.2026] C_contribution:
Aim: Determination of the symmetry elements and the point group of a molecule.

Rough sketch of the functioning of the program to be coded:
1. Identification of the point group (based on a decisional diagram -> to organise and somewhat limit unnecessary calculations)
    a. Check of the linearity
        i. Check for inversion centre
    b. Check planarity
    c. Check chirality ?
2. Deduction of the complete list of symmetry elements to find
3. Highlight of the symmetry elements on the molecule

Or 
1. Determination of all symmetry elements
2. Deduction of the point group of the molecule

Symmetry elements to consider:
- inversion centre i
- proper rotation axis C
- improper rotation axis S
- symmetry planes sigma (subclasses: h, v, d)

Major existing tools that may be implemented:
- use of data bases: pubchem
- computations with matrices supported through numpy
    - calculation based on cartesian coordinates
    - determination of the centre of mass (physical: atom weighted given the element, aphysical: all atoms having the same weight)

Organisation of the built tool: (in a python package repository)
- Main document corresponding to the Software
- Specific library of functions (1 module per symmetry element ?)
- Notebook of tests
- Report showcasing the functionalities
- Readme file with description, requirements and installation information
- Opensource license

Limits of the tool to be built:
- Computations performed only for the most stable form of the molecule, determined via rdkit functionalities.
- Elements of symmetry limited to the global symmetry of the molecule. No determination of the local symetries.

User interface:
- Several possible inputs:
    - SMILES
    (- IUPAC names)
    (- Drawing interface)

- Control on the generated outputs:
    - point group of the molecule
    - visual of the studied molecule with toggle key for each symmetry element highlight option (show/hide)
